# My E-Portfolio

A personal project where I showcase my work and learnings as a software engineer. This site showcases my professional and pesronal works, mainly built with Three.js, Framer-Motion and React-Remix.

---

## How to Run Locally

If you want to run this portfolio on your own machine:

1. **Clone the repository**

   ```bash
   git clone https://github.com/NironR/e-portfolio.git
   cd e-portfolio
   ```
2. **Install Dependecies**

   ```bash
   npm install
   ```

3. **Run the development server**
    ```bash
   npm run dev
   ```
